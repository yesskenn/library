# Web & UI Reading Shelf

A lightweight web app for organizing UX and web development -related reading resources. Built with vanilla JavaScript using state management, DOM manipulation, and dynamic UI updates.

## Features

- Add new reading resources
- Remove resources
- Mark resources as read or unread
- Dynamic interface updates
- Unique IDs for reliable data management

## Tech Stack

- HTML
- CSS
- JavaScript
